# О приложении

**Happ** — кроссплатформенное приложение для создания, хранения и управления пользовательскими профилями подключения, использующими конфигурации на базе технологии **Xray**.

Программа предназначена для работы на мобильных устройствах и настольных компьютерах и предоставляет единое рабочее пространство для настройки параметров соединения. Пользователь может создавать собственные профили, добавлять заранее подготовленные конфигурации, изменять отдельные параметры, сохранять различные варианты настроек и выбирать подходящий профиль для конкретных условий использования.

Архитектура Happ построена на основе независимых конфигурационных профилей. Каждый профиль представляет собой отдельный набор параметров, который может включать информацию о подключении, правила маршрутизации, настройки обработки сетевых потоков и дополнительные элементы управления. Раздельное хранение конфигураций позволяет удобно организовывать несколько вариантов настроек, управлять ими отдельно и быстро переключаться между ними при необходимости.

# Возможности работы с профилями

Happ предоставляет инструменты для полноценного управления конфигурациями:

* создание пользовательских профилей подключения;
* импорт настроек из файлов, сохранённых на устройстве;
* редактирование существующих конфигураций;
* хранение нескольких независимых наборов параметров;
* настройка правил маршрутизации;
* изменение дополнительных параметров соединения;
* выбор активного профиля;
* обновление ранее сохранённых настроек;
* выполнение всех операций в локальной среде устройства.

Каждая конфигурация хранится отдельно, что позволяет пользователю создавать собственную систему профилей для разных задач и вариантов использования.

# Хранение и защита пользовательских данных

Все созданные профили и импортированные конфигурационные файлы сохраняются исключительно в локальной памяти устройства.

Приложение не использует внешние облачные сервисы для хранения настроек, не выполняет автоматическую синхронизацию данных между устройствами и не передаёт информацию о пользовательских профилях сторонним системам.

Все операции с конфигурациями выполняются непосредственно пользователем внутри приложения: добавление, импорт, редактирование, сохранение, удаление и активация профилей осуществляются на выбранном устройстве.

Happ не анализирует содержимое импортируемых конфигурационных файлов, не использует содержащиеся в них данные для дополнительных целей и не предоставляет доступ к пользовательским настройкам третьим сторонам.

# Особенности функционирования

Happ представляет собой локальный инструмент управления параметрами подключения и не является поставщиком сетевой инфраструктуры.

Приложение не содержит:

* собственные VPN-серверы;
* прокси-сервисы;
* удалённые системы обработки или перенаправления данных;
* встроенные коллекции готовых конфигураций;
* автоматические механизмы получения профилей подключения.

Все параметры для работы с профилями пользователь добавляет самостоятельно — путём ручного ввода либо через импорт заранее подготовленных конфигурационных файлов.

Happ не устанавливает назначение пользовательских профилей и не ограничивает способы их применения. Пользователь самостоятельно отвечает за использование приложения в рамках законодательства и требований, действующих в его стране или регионе.
